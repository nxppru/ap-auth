openWMS

OpenWMS是集客科技开发的基于wifidog协议的认证营销平台。采用开源GPL许可证，任何个人和公司都可以免费下载、安装和使用本系统，没有限制，也没有许可费用。 OpenWMS使用PHP语言开发，可以运行在windows和linux系统上，也可以在阿里云等云服务器上部署。 OpenWMS支持与所有wifidog的路由器对接。 OpenWMS支持远程集中控制集客科技的系列路由系统（集客盒子和集客网关）。
